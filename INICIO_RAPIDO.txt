1️⃣  COMMIT DAS ALTERAÇÕES

   cd /Users/Projetos/dogCeo
   git add .
   git commit -m "feat: Adicionar configuração CI/CD completa"


2️⃣  CRIAR REPOSITÓRIO NO GITHUB

   • Acesse: https://github.com/new
   • Nome: dogCeo
   • NÃO marque "Initialize with README"
   • Clique em "Create repository"


3️⃣  CONECTAR E FAZER PUSH

   git remote add origin https://github.com/SEU_USUARIO/dogCeo.git
   git branch -M main
   git push -u origin main


4️⃣  CONFIGURAR PERMISSÕES (No GitHub)

   Settings → Actions → General → Workflow permissions
   → Selecione "Read and write permissions"
   → Marque "Allow GitHub Actions to create and approve pull requests"
   → Save


5️⃣  CONFIGURAR GITHUB PAGES (No GitHub)

   Settings → Pages → Source
   → Branch: gh-pages
   → Folder: / (root)
   → Save
